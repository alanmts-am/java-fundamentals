# Exercício 01 — Sistema de Gestão de Estoque

Crie um sistema de Gestão de Estoque que:

- Declare uma variável para o `nomeDoProduto`
- Declare o `precoUnitario`
- Declare a `quantidadeEmEstoque`
- Calcule o `valorTotalEmEstoque`
- Crie uma variável booleana chamada `precisaRepor` que será `true` se a quantidade for menor que 5
- Imprima todas essas informações no console de forma organizada

---

## Verificação

**Qual a diferença entre `int` e `double`?**
Int seria a respresentação dos inteiros enquanto double seria dos decimais, pontos flutuantes


**Por que usamos `String` (com S maiúsculo) e `int` (com i minúsculo)?**
Int é um tipo primitivo que herda de Integer. Já a String é um objeto, tanto que podemos chamar métodos dela

> Dica: Pense em quem é um objeto e quem é um tipo primitivo.
